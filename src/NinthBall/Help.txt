Usage:
 {me} --in Input-config-path-and-glob --watch
 {me} --help

Switches:
 --help         Display this help text.
 --watch        Watch the input files for changes and automatically reprocess.

Parameters:
 --in           Path to the simulation input, a folder, an YAML file or glob pattern.
 --out          Path to an optional simulation output configurations YAML file.
