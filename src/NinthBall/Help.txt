
Usage:
 {me} --in Input.yaml --out Output.html --watch
 {me} --sampleyamls
 {me} --help

Switches:
 --help         Display this help text.
 --sampleyamls  Write sample configuration files (Input & Output) to the current directory.
 --watch        Watch the input file for changes and automatically reprocess it.

Parameters:
 --in           Path to the input YAML file.
 --out          Path to the output HTML file. Defaults to <input>.html if not specified.
