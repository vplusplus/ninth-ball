
# Monte Carlo Financial planning simulation year-by-year cashflow illustration

This dataset contains year-by-year cashflow illustration of a Monte Carlo financial planning simulation.
All values are synthetic and contain **no PII**.

## About the data set

### `Input`

Represents a scenario and constraints chosen by the user.

### `Assumptions`

Represents the financial moodeling assumptions simulated by Monte Carlo simulation.

### `ResultSummary'

High level summary of 10,000 iterations. 
Provides key metrics for select percentiles.

### `YearByYear??thPercentile`

You will see mutiple elements, one for each percentile.
Each node provides year by year cashflow, market performance, and inflation.
